# CrousBot

## Installation

Create a virtualenv.

Install requirements:
```pip install -r requirements.txt```

Create a .env file and fill it.

## Launching

Start bot script:
```python main.py```

## Docker

Simple docker deployment:

Create a virtualenv.

Then create the container using docker-compose:
```docker compose up```
